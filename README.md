

A full-stack login and registration system built using **Spring Boot (backend)**, **React (frontend)**, and **PLSQL (database)**.  
This project allows users to register, log in, and securely store credentials in the database.

---

##  Features

- User Registration with validation  
- Secure Login using Spring Security  
- Password encryption with BCrypt  
- RESTful API for authentication  
- Persistent user data stored in MySQL  
- CORS-enabled for frontend communication  


---

##  Tech Stack

**Backend:** Java, Spring Boot, Spring Security, JPA, Hibernate  
**Frontend:** React.js  
**Database:** MySQL  
**Build Tool:** Maven  
**IDE:** Eclipse  
**Server:** Tomcat (embedded)
